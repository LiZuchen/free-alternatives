# 🤝 贡献指南

欢迎为 **Free Alternatives** 项目做出贡献！

## 📝 如何贡献

### 1. 提交新的替代方案

发现新的免费替代方案？欢迎添加！

**步骤**：
1. Fork 本项目
2. 在 `README.md` 中添加新内容
3. 提交 Pull Request

**格式模板**：
```markdown
### 收费软件 → 🆓 免费替代

| 特性 | 收费软件 | 免费替代 |
|------|---------|---------|
| 价格 | ¥XX/月 | 免费 |
| 平台 | Win/Mac | Win/Mac/Linux |
| 功能 X | ✅ | ✅ |

**推荐场景**：
- 场景 1 → **替代方案**

**下载**：[官网链接]()
**教程**：[学习资源]()
```

---

### 2. 修正错误

发现信息有误？欢迎指正！

- 价格变动
- 链接失效
- 功能描述错误
- 新增更好的替代方案

直接提交 Issue 或 Pull Request 即可。

---

### 3. 翻译

帮助翻译成其他语言？太棒了！

当前需要：
- 🇺🇸 英文版
- 🇯🇵 日文版
- 🇰🇷 韩文版

创建对应语言的 README 文件即可。

---

## 📋 审核标准

提交的内容需要满足：

- ✅ 确实是免费/开源软件
- ✅ 有官方网站或可靠下载源
- ✅ 信息准确（价格、功能等）
- ✅ 无恶意软件/广告软件
- ✅ 有实际使用价值

---

## 🎨 代码风格

- 使用 Markdown 表格对比
- 添加 emoji 增加可读性
- 保持客观中立
- 标注信息来源
